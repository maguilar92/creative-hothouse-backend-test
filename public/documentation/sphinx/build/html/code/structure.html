Structure¶

The project is structured as follows:

  • Modules
    • Core
    • Cryptocurrency
    • User
  • Themes
    • Www

Modules¶

Here we will find the logic of our application, or what is the same, the backend.

Core¶

This module is responsible for loading all the common used by other modules and the views of the frontend that use vuejs.

Cryptocurrency¶

This module is used for the visualization and management of cryptocurrencies.

User¶

This module is used to authenticate with the application.

Themes¶

Here we will find the frontoffice of our application.

Www¶

This theme is the used for all the public with what the users interact.
